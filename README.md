#### 01、轻松保存网页文件(图片和视频)到手机和平板，稍后阅读。
#### 02、保存的网页提供优化阅读功能，自动排版功能，优雅的字体，可调节文字大小，为您提供愉悦的阅读体验。
#### 03、支持知乎日报订阅和阅读。
#### 04、支持保存网页长图片、网页另存为、网址黑名单。
#### 05、支持标签管理，可以使用 #独乐乐# 标签隐藏指定离线文章。
#### 06、支持RSS订阅。
#### 07、支持文件快传，非常方便的在手机和电脑分享文件，手机之间也行，安卓和苹果也可。
#### 08、提供图片墙，像浏览网页一样查看本地图片和视频。
